准备工作#

1.关于GitHub的基本用法#

参考：https://pwner.cn/posts/16fb9e75.html

2.关于Pycharm的安装#

官网下载安装即可：https://www.jetbrains.com/Pycharm/

3.关于Git的安装#

官网下载安装即可：https://Git-scm.com/download/win

安装好 Git 后可以在本地检查

Pycharm中配置GitHub的方法#

1.在Pycharm中检查Git环境是否安装完毕#

检测显示Git版本说明Git环境已经成功配置，若未配置好Git环境请自行百度安装（安装过程比较简单，下载好安装包后一路下一步即可）

一般都可以自动检测到Git的安装路径，如果没有自动检测到，请手动选择Git的安装路径

2.在Pycharm中配置GitHub账号信息#

这样就完成了Pycharm和GitHub的账号关联，接下来就可以在Pycharm中使用GitHub进行项目的管理了

3.本地项目文件夹和GitHub远程仓库的关联#

建立一个本地项目文件夹#

假设这是本次需要进行版本管理的项目根目录

使用

git init

命令初始化项目

创建GitHub远程仓库#

在GitHub创建同名仓库便于管理

将本地文件夹和远程仓库关联#

git remote add origin Git@github.com:PwnerZhang/local2021.Git

将项目文件托管到GitHub#
